# UniversalClipboard

UniversalClipboard is a cross-device clipboard synchronization application.

## Current Progress
- Android app initialized with Jetpack Compose
- Clipboard history UI implemented
- Local Room database for clipboard storage
- Dependency and Gradle configuration completed

## Tech Stack
- Kotlin
- Jetpack Compose
- Room Database
- Android SDK
